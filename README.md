# cors-sample

CORS sample code.

```bash
$ cd ~/GOPATH/your_path
$ git clone git@github.com:mochizukikotaro/cors-sample.git && cd $(basename $_ .git)

$ realize start
```

```bash
$ curl http://localhost:8888
{"status":200,"result":"Cookie が取得できなかった"}%
```

```bash
$ cd ~your_path/cors-sample/php
$ php -S localhost:9999
# => http://localhost:9999
```
